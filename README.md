Electricity Consumption Analysis Using Tableau 
Project Overview

Plugging into the Future – Electricity Consumption Analysis Using Tableau is a data visualization project that analyzes electricity consumption patterns across Indian states and regions for the years 2019 and 2020. The project transforms raw electricity consumption data into interactive dashboards, enabling stakeholders to identify trends, compare regional usage, and evaluate the impact of events such as COVID-19 lockdowns.




Key Features
Interactive state-wise and region-wise electricity consumption analysis
Monthly, quarterly, and yearly trend visualization
Lockdown impact analysis on electricity demand
Top and Bottom electricity-consuming states ranking
Dynamic filters for Year, Region, State, and Quarter
KPI cards and storytelling dashboards for decision-making
Exportable reports and dashboard sharing options
